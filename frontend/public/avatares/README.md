# Avatares de Usuario

Este directorio contiene las imágenes de avatar por defecto para los usuarios del sistema.

## Estructura

- `masculino.png` - Avatar por defecto para usuarios con género masculino
- `femenino.png` - Avatar por defecto para usuarios con género femenino  
- `otro.png` - Avatar por defecto para usuarios con género otro o no especificado

## Uso

El sistema selecciona automáticamente el avatar correspondiente según el género del empleado registrado en la base de datos.

Los avatares se muestran en:
- Navbar (menú de usuario)
- Perfil de usuario

## Formato

- Formato: PNG
- Tamaño recomendado: 200x200 píxeles
- Fondo: Transparente
